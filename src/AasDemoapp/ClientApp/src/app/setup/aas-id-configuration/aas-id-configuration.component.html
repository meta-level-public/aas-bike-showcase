AAS ID-Präfix
AAS ID-Präfix konfigurieren

Konfigurieren Sie hier den URL-Präfix, der für die Generierung von AAS IDs verwendet wird. Dieser Präfix wird allen generierten Asset Administration Shell IDs, Submodel IDs und Asset IDs vorangestellt.

Dieser Wert wird für die Generierung aller IDs in der AAS-Umgebung verwendet.
Beispiel einer generierten ID

{{ aasIdPrefix() }}/ids/aas/1234_5678_9012_3456
